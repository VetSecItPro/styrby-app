# Styrby

> Real-time budget tracking for AWS and cloud costs

**Live Site:** [https://styrby.dev](https://styrby.dev)

Styrby helps developers and teams monitor cloud spending in real-time with instant alerts and beautiful dashboards. Stop cloud bill surprises.

## Tech Stack

- Next.js 15 / React 19
- TypeScript
- Supabase
- CLI (Node.js)
